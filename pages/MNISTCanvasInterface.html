canvas

The interface above lets you draw a number in the left-hand box and send it to a neural network to be recognized. The result will appear in the right-hand box. The app is running over at Heroku; if the interface above doesn't seem to be working, try the interface hosted on the same server.

The back-end is a Python Flask application, using keras to load up a pregenerated neural network trained on the MNIST dataset. For more information, see the app's GitHub repository.
